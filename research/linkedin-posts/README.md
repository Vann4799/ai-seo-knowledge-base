# LinkedIn Posts

This folder contains manually collected public LinkedIn post records organized by expert.

Each expert folder includes:

- Direct LinkedIn post links.
- Date shown during collection.
- Short summary of the post.
- Why the post is relevant to AI-powered SEO content production.

The project intentionally stores short research notes and links rather than full copied LinkedIn post text. This keeps the repository readable and avoids copying long platform content verbatim.

See `research/linkedin-posts/manifest.csv` for a consolidated list of collected post records.
